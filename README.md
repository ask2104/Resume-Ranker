# Resume Ranker using NLP

Resume Ranker is an automated screening tool designed to streamline the recruitment process. By leveraging Natural Language Processing (NLP) and Machine Learning, the system evaluates resumes against job descriptions to identify the most suitable candidates for a specific role.

## Project Description

Developed a Gradient Boosting Model to automate the process of resume screening by using NLP and TF-IDF feature classification in Google Colab.

It classifies the resume to find the best fit for the description or features of the role. A dashboard was created using Streamlit for ease of usage and understanding.

## Features

- Automated Screening: Rapidly evaluates multiple resumes against a target job description.
- NLP Integration: Utilizes Natural Language Processing to understand and categorize resume content.
- Fit Classification: Uses a Gradient Boosting Model to rank candidates based on role-specific features.
- Interactive Dashboard: A user-friendly Streamlit interface for uploading files and visualizing results.

## Tech Stack

- Language: Python 3
- Libraries: scikit-learn (Gradient Boosting), NumPy, Pandas
- NLP Tools: TF-IDF Vectorizer, NLTK/Spacy
- Frontend: Streamlit
- Environment: Google Colab / Local Python environment

## Installation and Setup

1. Clone the repository:
   git clone https://github.com/ask2104/Resume-Ranker.git
   cd Resume-Ranker

2. Install dependencies:
   pip install -r requirements.txt

3. Run the Dashboard:
   streamlit run app.py

## Usage

1. Open the Streamlit dashboard in your browser.
2. Upload one or more resumes in PDF or Text format.
3. Input the Job Description or key features required for the role.
4. View the ranked results and classification fit generated by the Gradient Boosting Model.
